CSS Box Model

The CSS Box Model is a fundamental concept that describes how elements are structured and spaced in a web page. It consists of four main parts: content, padding, border, and margin.

Box Model Components

  • Content: The area where text and images appear.
  • Padding: Space between the content and the border. It is transparent and expands the element's size.
  • Border: A line surrounding the padding (or content if no padding is set).
  • Margin: Space outside the border, separating the element from others.
.box {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

Box-Sizing Property

The box-sizing property controls how the total width and height of an element are calculated:

  • content-box (default): Width and height include only the content. Padding and border are added outside.
  • border-box: Width and height include content, padding, and border.
.box-content {
  box-sizing: content-box;
}

.box-border {
  box-sizing: border-box;
}
Content-Box

Total width = content + padding + border

Border-Box

Total width = specified width

Use Cases

  • Use padding to create space inside an element, such as between text and its border.
  • Use margin to create space between elements, such as separating cards in a grid.
  • Use box-sizing: border-box for consistent sizing in responsive designs.

Gotchas

  • Collapsing Margins: Adjacent vertical margins between elements may collapse into a single margin.
  • Padding and Border: When using content-box, padding and border increase the total size of the element, which can break layouts if not accounted for.
  • Inline Elements: Padding and border apply to inline elements but do not affect their flow in the document.
