CSS Grid

CSS Grid is a two-dimensional layout system for building complex, responsive layouts using rows and columns. It complements Flexbox, which is better for one-dimensional layouts.

Grid Basics

Create a grid container with display: grid and define columns/rows using grid-template-columns and grid-template-rows. 

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.item-tall { grid-row: span 2; }

Named Areas

Use grid-template-areas to map grid cells to names and assign items using grid-area. 

.layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-areas:
    "sidebar main"
    "sidebar footer";
}
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Responsive Patterns

Use repeat(), minmax(), and auto-fill/auto-fit for responsive grids.

.responsive {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
  gap: 10px;
}

Implicit vs Explicit Grids

Explicit grid tracks are those you define with grid-template-*. Implicit tracks are created automatically when items exceed the explicit grid; control them with grid-auto-rows and grid-auto-columns. 

.auto { display:grid; grid-template-columns: 1fr 1fr; grid-auto-rows: 80px; }

Alignment

Control alignment of items and the grid itself using:

  • justify-items / align-items (alignment inside cells)
  • justify-content / align-content (alignment of the whole grid within its container)
.align { display:grid; grid-template-columns: repeat(3, 1fr); justify-items:center; align-items:center; height:120px; }

Grid Templates

The grid-template-columns and grid-template-rows properties define the structure of the grid. You can specify sizes for each column or row using various units.

.grid {
  display: grid;
  grid-template-columns: 100px 1fr auto;
  grid-template-rows: 50px 2fr;
}

Units for Grid Templates

Grid templates support various units to define track sizes:

  • Pixels (px): Fixed size (e.g., 100px).
  • Percentage (%): Relative to the grid container (e.g., 50%).
  • Fractional Unit (fr): Distributes available space (e.g., 1fr, 2fr).
  • Auto: Automatically adjusts to the content size.
  • Minmax: Sets a minimum and maximum size (e.g., minmax(100px, 1fr)). 
.grid {
  display: grid;
  grid-template-columns: 100px 50% 1fr auto minmax(100px, 200px);
}

Useful Functions & Units

  • fr: Fractional unit for distributing available space.
  • minmax(min, max): Constrains track sizes.
  • repeat(): Repeats column/row patterns.
  • auto-fit / auto-fill: Useful with minmax() for responsive grids.

Best Practices

  • Use Grid for two-dimensional layouts (rows and columns).
  • Combine Grid and Flexbox: Grid for layout, Flexbox for internal alignment.
  • Prefer fr and minmax() for responsive sizing.
  • Use named areas for readable template layouts but avoid over-complication for small grids.
  • Test across browsers and use devtools grid inspector to debug layouts.
