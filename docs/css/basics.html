CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. This page introduces the foundational concepts of CSS for beginners.

What is CSS?

CSS is a stylesheet language that describes how HTML elements are displayed on the screen, paper, or other media. It allows you to control the layout, colors, fonts, and more.

CSS Syntax

CSS rules are made up of selectors and declarations. A declaration consists of a property and a value. 

selector {
  property: value;
}

Example:

h1 {
  color: blue;
  font-size: 24px;
}

The Cascade

CSS stands for Cascading Style Sheets, and the cascade determines which styles are applied when multiple rules match the same element. The cascade considers:

  • Importance (e.g., !important declarations).
  • Specificity (e.g., #id is more specific than .class).
  • Source order (later rules override earlier ones).

How to Add CSS

There are three ways to add CSS to a web page:

  • Inline: Add styles directly to an HTML element using the style attribute.
  • Internal: Use a <style> block inside the <head> of the HTML document.
  • External: Link to an external CSS file using the <link> tag. 
<link rel="stylesheet" href="styles.css">

Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree. It consists of:

  • Content: The actual content of the element.
  • Padding: Space between the content and the border.
  • Border: A border surrounding the padding (if any).
  • Margin: Space between the element and its neighbors.

Example:

div {
  margin: 10px;
  padding: 20px;
  border: 1px solid black;
}

Best Practices

  • Use external stylesheets for better maintainability.
  • Organize your CSS with comments and logical groupings.
  • Use shorthand properties (e.g., margin: 10px 20px;).
  • Test your styles across different browsers and devices.
