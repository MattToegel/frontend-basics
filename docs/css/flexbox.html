CSS Flexbox

Flexbox is a CSS layout model that provides an efficient way to align and distribute space among items in a container, even when their sizes are dynamic. This page covers the key concepts and properties of Flexbox with visual examples.

Flex Container Properties

The following properties are applied to the flex container:

  • display: flex: Defines a flex container.
  • flex-direction: Specifies the direction of the main axis (row, row-reverse, column, column-reverse).
  • justify-content: Aligns items along the main axis (flex-start, center, space-between, space-around, space-evenly).
  • align-items: Aligns items along the cross axis (flex-start, center, flex-end, stretch, baseline).
  • align-content: Aligns a flex container's lines when there are multiple lines (flex-start, center, space-between, space-around, stretch).

Flex Item Properties

The following properties are applied to flex items:

  • flex-grow: Specifies how much a flex item will grow relative to the rest.
  • flex-shrink: Specifies how much a flex item will shrink relative to the rest.
  • flex-basis: Specifies the initial size of a flex item before space distribution.
  • align-self: Overrides the align-items property for a specific item (auto, flex-start, center, flex-end, stretch).

Best Practices

  • Use Flexbox for one-dimensional layouts (either row or column).
  • Combine Flexbox with Grid for complex layouts.
  • Test your layout on different screen sizes to ensure responsiveness.
  • Use gap for spacing between flex items instead of margins for cleaner layouts.

Shorthand & common patterns

The flex shorthand combines flex-grow, flex-shrink, and flex-basis. A common pattern is flex: 1 which allows items to grow equally. 

/* Examples */
.container { display: flex; }
.container .item { flex: 1; } /* shorthand for equal growth */
.container .item-wide { flex: 2; } /* twice the width */
.container .item-fixed { flex: 0 0 100px; } /* fixed width */

Wrapping & multi-line containers

Use flex-wrap: wrap to allow items to wrap to the next line. When wrapping, align-content controls the cross-axis alignment of the rows. 

.wrap { display:flex; flex-wrap:wrap; width:300px; }

Order & per-item alignment

The order property changes the visual order of items without changing the DOM. Use align-self to override an item's alignment. 

.item-a { order: -1; } /* appear first */
.item-b { align-self: flex-end; } /* override container align-items */

Centering and gap

Centering both axes is simple with flexbox: use justify-content and align-items. Use gap for spacing between items. 

.center { display:flex; justify-content:center; align-items:center; gap:10px; height:150px; }

Flex-basis & equal distribution gotcha

To distribute space equally regardless of intrinsic content size, use flex-basis: 0 in combination with flex-grow (e.g., flex: 1 1 0).

/* Ensures equal widths */
.equal > div { flex: 1 1 0; }

Notes & tips

  • Flexbox is ideal for one-dimensional layouts; use Grid for two-dimensional layouts.
  • Use gap instead of margins for cleaner spacing between flex items.
  • Remember align-content affects multi-line containers; align-items affects single-line alignment.
  • Test flex layouts on narrow screens and older browsers if you need broad compatibility.
