CSS Selectors

CSS selectors are patterns used to select and style specific elements in an HTML document. This page covers the most common types of selectors with examples and explanations.

Basic Selectors

  • Universal Selector (*): Selects all elements.
  • Type Selector: Selects all elements of a specific type (e.g., p, h1).
  • Class Selector (.classname): Selects all elements with a specific class.
  • ID Selector (#id): Selects a single element with a specific ID.
  • Group Selector (selector1, selector2): Selects multiple elements.
/* Examples */
* {
  margin: 0;
}

p {
  color: blue;
}

.button {
  background-color: green;
}

#header {
  font-size: 24px;
}

h1, h2 {
  font-family: Arial, sans-serif;
}

Combinator Selectors

  • Descendant Selector (ancestor descendant): Selects all descendants of an element.
  • Child Selector (parent > child): Selects direct children of an element.
  • Adjacent Sibling Selector (element1 + element2): Selects the next sibling of an element.
  • General Sibling Selector (element1 ~ element2): Selects all siblings of an element.
/* Examples */
div p {
  color: red; /* All <p> inside <div> */
}

div > p {
  font-weight: bold; /* Direct children <p> of <div> */
}

h1 + p {
  margin-top: 10px; /* First <p> after <h1> */
}

h1 ~ p {
  color: gray; /* All <p> after <h1> */
}

Pseudo-classes

Pseudo-classes define the special state of an element.

  • :hover: Applies when the user hovers over an element.
  • :nth-child(n): Selects the nth child of a parent.
  • :first-child: Selects the first child of a parent.
  • :last-child: Selects the last child of a parent.
  • :not(selector): Excludes elements matching the selector.
/* Examples */
a:hover {
  text-decoration: underline;
}

li:nth-child(2) {
  color: orange; /* Second <li> */
}

p:first-child {
  font-style: italic; /* First <p> in a parent */
}

p:not(.highlight) {
  color: black; /* All <p> except those with class "highlight" */
}

Pseudo-elements

Pseudo-elements style specific parts of an element.

  • ::before: Inserts content before an element.
  • ::after: Inserts content after an element.
  • ::first-line: Styles the first line of text in an element.
  • ::first-letter: Styles the first letter of text in an element.
/* Examples */
p::before {
  content: "Note: ";
  font-weight: bold;
}

p::after {
  content: " [Read more]";
  color: blue;
}

p::first-line {
  font-size: 1.2em;
}

p::first-letter {
  font-size: 2em;
  color: red;
}

Best Practices

  • Use class selectors for reusable styles.
  • Minimize the use of ID selectors to avoid specificity issues.
  • Organize your CSS with comments and logical groupings.
  • Test your selectors to ensure they target the intended elements.
